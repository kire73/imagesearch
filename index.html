Image Search Abstraction layer


Search Images by adding to the URL:


https://picsee.herokuapp.com/api/imagesearch/pictures of cats


Paginate your search results by including the option:

?offset=10
Like this:
https://picsee.herokuapp.com/api/imagesearch/funny?offset=8
Will default to 5 results per search if not set
